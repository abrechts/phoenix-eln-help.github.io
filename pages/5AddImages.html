5) Add Images

Sometimes an image can say more than a thousand words, so here are the options for inserting an image into your protocol: 

Insert via Drag & Drop

The most convenient way for adding an image or document is to drag one or multiple files from the desktop or the file explorer and to drop it into the desired protocol location:

.

Insert via File Dialog

Alternatively, images and documents can be inserted via a file selection dialog. To do this, choose the Images/Documents menu item in the Addition Toolbar and select the desired image(s) and document(s) to be imported. The file selection dialog allows the selection of multiple items.

Imported Images

The protocol handles image and document file types differently. Let's have a look a images first. After their import, they are represented as fixed height thumbnail images with an editable image description area to their right.
